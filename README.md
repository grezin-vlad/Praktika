# Классификация спутниковых изображений EuroSAT

Проект выполнен в рамках производственной практики.

## Используемые архитектуры

- ResNet18
- ResNet50
- MobileNetV3
- EfficientNet-B0
- DenseNet121

## Лучшая модель

EfficientNet-B0

Accuracy: 98.52%

## Запуск

pip install -r requirements.txt

python src/demo_app.py

или

streamlit run src/streamlit_app.py
